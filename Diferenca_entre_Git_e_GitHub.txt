Qual a diferença entre Git é GitHub?
Git vs Github: Qual é a Diferença e Como Começar com Ambos
git é um software VCS local que permite aos desenvolvedores salvar snapshots de seus projetos ao longo do tempo. Geralmente é melhor para uso individual. GitHub é uma plataforma baseada na web que incorpora os recursos de controle de versões do git para que possam ser usados colaborativamente.
